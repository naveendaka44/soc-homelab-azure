# Azure SOC Home Lab

## Objective
Built a cloud-based Security Operations Center (SOC) using Microsoft Azure 
to simulate real-world threat detection, incident response, and SIEM operations.

## Tools Used
- Microsoft Azure (Free/Pay-as-you-go)
- Microsoft Sentinel (SIEM/SOAR)
- Log Analytics Workspace
- Microsoft Defender for Cloud
- Azure Virtual Machines (Windows & Linux)
- KQL (Kusto Query Language)
